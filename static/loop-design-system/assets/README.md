## 사용 가이드

### 웹사이트 favicon 적용
```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/loop-icon-gradient.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/loop-icon-192x192.png">
```

### PWA manifest.json
```json
{
  "icons": [
    { "src": "loop-icon-192x192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "loop-icon-512x512.png", "sizes": "512x512", "type": "image/png" }
  ]
}
```

### 로고 선택 가이드
- **밝은 배경**: `loop-logo-gradient` 또는 `loop-logo-flat`
- **다크 배경**: `loop-logo-white`
- **이메일/문서**: `loop-logo-flat` (단색, 호환성 최상)
- **고급 마케팅 자료**: `loop-logo-gradient`
